=head1 FEC PARSER

Converting FEC contribution data from the COBOL format to CSV for loading into MySQL.

=head2 VERSION

version 0.1a (DEPRECIATED)

=head2 SYNOPSIS

FEC Data is in a stupid format. This converts it into a normalized CSV files for upload in MySQL.

=head2 INSTALLATION

1) Download the necessary files via git clone.

2) Change a lot of the absolute file paths.

*NOTE: Very quickly written a long while back, YMMV.

=head2 DEPENDENCIES

All should be available with Perl or CPAN.

Must load Archive::Extract
